# Pathment Suggestions

A shared, living list of suggestions for Pathment, and what we have covered so
far. This is meant to be a single sheet that anyone can read, comment on, and
add to. Drop a comment next to anything, or add your own idea at the bottom.

Nothing here is final. It is a working list, not a spec.

Quick key: Done means it is already in the prototype. Idea means it is on the
list but not built yet.

## What we have covered so far

### Mentor setup and guidance

- Mentor onboarding flow, a short first-run setup for profile, AI key, a
  schedule to adopt, and 1:1 availability. Done
- Mentor spec and FAQs, a reference for what a mentor does and how. Done

### The day, the talks, and the schedule

- Three talks, three meals: Mindset talk at breakfast, Engineering talk at
  lunch, Dean talk at dinner. Done
- Morning journaling and morning reading at the start of the day. Done
- Weekday structure versus the weekend, where the weekend is just the long grind
  plus family time. Done
- Talks are patched onto meals on purpose, this is habit stacking, the meal is
  the anchor that makes the talk automatic. Done
- Build, assign, and edit custom schedules, and fill each slot per mentee with a
  roadmap or a recurring task. Done
- Roadmaps as templates that auto-assign tasks, roadmap chains, and recurring
  tasks. Done
- A simpler way to link tasks to each slot per mentee, one clean editor instead
  of several. Done

### Reports and communication

- Weekly and monthly reports generated by AI. Done
- Weekly encouraging emails based on stats, for example the best member from each
  clan or fellowship. Done
- Org announcements, one channel to move off scattered chats like WhatsApp. Done
- Checklists for specific kinds of tasks, for example an article, branding, or
  notes. Idea, partly covered by task approval criteria today.

### Specialized clans

- Clans organized by technology. Done
- Clans organized by level. Done
- Communities inside a clan. Done

### Gamification

- Leaderboard. Done
- XP, levels, and streaks. Done
- Gifts. Done
- Incentives. Done
- Referrals. Done
- Promotion into a talent network. Done
- Co-mentees for specific access. Idea, partly covered by collaborators and
  cross-clan access today.

### The human level

- Progress score with a chosen time window. Done
- Absolute versus relative progress, measured fairly. Done
- Daily logs with a quick note per item. Done
- Personality report from the mentor and from AI. Done
- Mentor report and feedback. Done
- Inviting a psychologist or specialist, with everything they log attributed to
  them. Done
- Sharing and controlling the personality read, used only to help. Done

### Cross-clan and governance

- Clan change requests, with a reason chosen from a dropdown. Done
- Cross-clan assignments, including additional responsibility for a clan and
  co-mentee access. Done
- Policies that govern clan changes, cross-clan work, and promotion. Done

### Mentee experience

- A community feed for mentees, peer kudos, wins, questions, and light internal
  memes. Done
- Mentees motivating other mentees. Done
- A My Mentees view so a mentor can see all of their mentees in one place. Done
- 1:1 availability the mentor publishes, plus a way for a mentee to request a
  1:1. Done

### Growing the next mentors

- Promotion to mentor or co-mentor. Done
- A brief interview, including how willing the person is to help others. Done
- Approval and promotion. Done

### Polish and consistency

- A clean editorial style, no soft glows or gradients. Done
- Removed em-dashes and AI-ish characters from all copy. Done
- Fixed card padding so text is not flush to the edges, and fixed mentee-facing
  copy that read like mentor-facing copy. Done

## New ideas and comments

Add your suggestions here. A line or two is enough. Anyone can pick one up later.

- Group the long mentor sidebar into sections, for example Core, Growth, and
  Org, so it is not one long list.
- Let a mentor fill a slot with the same roadmap or recurring task for several
  mentees at once, not one at a time.
- Persist community posts, announcements, and promotions so they survive a
  reload and can drive real notifications and emails.
